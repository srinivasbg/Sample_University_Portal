# Sample_University_Portal
This is sample university portal written in HTML, PHP, SQL and JASON for fun. Provides the following functions:


For Students:


User authentication.
User management.
Current Standing.
Course Enrollement.


For Instructor:


Number of students enrolled.
Number of students in waiting list.
Courses and Section Info.
Student enrollment information.
Grading.


Please use the sample sql queries to get the test database.
